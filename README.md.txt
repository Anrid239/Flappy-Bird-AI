# Flappy Bird AI

Code to train an AI that can play "Flappy Bird" forever.


# Note - 

Before running flappybird.py, first if you have all the modules (pip files) installed, namely - pygame, matplotlib, neat, graphviz.
Also save the file visualize.py and config-feedforward.txt in the same folder as flappybird.py.
Enter the path to images carefully in the flappybird.py script, otherwise the game won't run.
Run flappybird.py to start the simulation
